

- SalesRegistration.py:  interacts with the user to gather a product
  name, price and quantity. Each input is validated (non-empty name,
  non-negative numeric price, non-negative integer quantity). The
  function returns the data as a tuple plus the computed total cost.

- ContinueShopping.py:  contains the main loop.  It repeatedly asks the
  customer if they want to buy something.  "Yes" triggers a call to
  SalesRegistration; the result is added to a purchase history.  "No"
  ends the loop and prints a receipt summarizing every item and the
  grand total.  Inputs other than "yes"/"no" produce an error message.

- main.py:  simply imports and calls ContinueShopping, starting the
  interactive shopping session.
